# "Want You Gone" Terminal Python Recreation

Este mini-projeto tem o foco de recriar o terminal dos creditos de Portal 2, se utilizando de Python para emular o DOS da música "Want You Gone". 

Este projeto tem a finalidade de emular a experiencia se os creditos em si funcionassem em um terminal, como se fosse um computador antigo.


## Tecnologias Usadas

- Python
- os
- pygame
- time
- sys
- ASCII Art
- pyinstaller

## Começando
### Pré-requisitos
- Interpretador Python (VS-Code)
- Bibliotecas:
    - os
    - time
    - pygame
    - sys
### Instalação
1. Clone o repositorio para sua máquina local ou baixe diretamente o .zip
2. Vá para o diretorio

### Uso
BAIXE a biblioteca pygame, ela é essencial para o funcionamento do codigo


## Aprendizados

O projeto foi direcionado, como uma forma de entender funções de delays e o uso de midias externas, no caso a música (.mp3). Este projeto teve apenas o foco criativo e de aprendizado, ou seja, nada comercial.
